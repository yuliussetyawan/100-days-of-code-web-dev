## Day 67: File Uploads & Error Handling in ExpressJS

Welcome to Day 67 of my 100 Days Of Code journey in the "2023 Web Development Bootcamp." Today's focus is on handling file uploads with multer in ExpressJS.

**Day 67 Highlights:**

- **The Two Sides Of Uploading Files**: Discussed the dual perspectives of uploading files and initiated a starting project for hands-on practice.

- **Adding A File Picker To A Form**: Implemented a file picker in a form, laying the groundwork for file uploads.

- **Parsing Incoming File Uploads With "multer" Package**: Introduced the "multer" package for parsing incoming file uploads, a crucial step in the process.

- **How To Store Files On A Backend**: Explored methods to store files on the backend, ensuring a seamless file upload experience.

- **Configuring Multer In-Depth**: Delved into an in-depth configuration of Multer, gaining a comprehensive understanding of its functionality.
